
C++ program:

Каждая кривая должна иметь возможность возвращать трехмерную точку и первую производную (трехмерный вектор) для каждого параметра t вдоль кривой.

Reqirements to implementation.

2. Has neither explicit memory deallocation nor memory leaks.

2. Не имеет ни явного освобождения памяти, ни утечек памяти.
